# Specs index

Specs for elements in the theme ‚Äî shipped (retrofit) and planned. Organized by composition layer (see `.context/docs/composition-strategy.md`). Each spec describes the API, output shape, behavior, and validation surface per `validation-contract.md`.

Originally sourced from the EXPLORATION-2 inventory pass (May 2026); retrofits added as shipped elements get documented.

## Substrate

- [theme-color](./theme-color.md) ‚Äî named color tokens emitted as `--color-<handle>` in `:root`; scheme-role tokens (`--color-role-<role>`) live in a disjoint namespace *(shipped ‚Äî retrofit)*
- [utility--css-variables](./utility--css-variables.md) ‚Äî the design-system CSS variable emitter; five domains (theme palette, gradients, color-scheme role tokens, text-style typography + base aliases, gutter spacing) emitted into `:root` and per-scheme rule blocks *(shipped ‚Äî retrofit)*
- [utility--color-contrast](./utility--color-contrast.md) ‚Äî two-choice contrast picker wrapping Liquid's `color_contrast` filter; echoes the higher-contrast reference between two candidates against a background *(shipped ‚Äî retrofit)*
- [utility--block-layout-vars](./utility--block-layout-vars.md) ‚Äî centralized emitter for the three per-instance vars shared by every L1 block (`--content-width`, `--mobile-margin-block-start`, `--desktop-margin-block-start`); skip-on-default logic + px‚Üírem conversion; consumed by all 9 shipped L1 block snippets via their `dynamic_style` capture *(shipped ‚Äî retrofit)*
- utility--media-sizing ‚Äî shared resolver for the `media_size` metaobject's mode dispatch (`ratio` / `relative` / `fixed` types + handle-routed `fill`); centralizes the mode-to-CSS emission logic so `media` and `embed` snippets share the resolution *(planned)*
- [base-component](./base-component.md) ‚Äî custom-element foundation class extending `HTMLElement`; lazy-instantiates the four manager subsystems (events, observers, cache, modifiers) and auto-clears them on disconnect; registered directly as `<token-section>` and subclassed by future specialized sections *(shipped ‚Äî retrofit)*
- [text-style](./text-style.md) ‚Äî reusable typography style metaobject; entries emit `--<handle>-font-*` variables + a selector rule binding them; `h1`‚Äì`h6` handles auto-bind to bare tags; one entry (`base_text_style` setting) emits `--base-*` aliases consumed by the body rule *(shipped ‚Äî retrofit)*
- [gradient](./gradient.md) ‚Äî scheme-adaptive linear gradient metaobject; angle + two scheme-role endpoints emit one `--gradient-<handle>` declaration whose stops re-resolve per color scheme; `background` handle reserved for the color-scheme system *(shipped ‚Äî retrofit)*
- [design-constants](./design-constants.md) ‚Äî scheme-independent CSS-variable constants in `layer-base.css` (7 scales: z-index, motion durations, motion easings, focus-ring metrics, border-radius, border-width, T-shirt spacing defaults) + AA touch-target floor (`--touch-target-min`) + the cascade-layer declaration + cross-document view-transition opt-in. Spacing defaults are auto-overrideable by matching `spacing` metaobject entries via cascade position *(shipped ‚Äî retrofit)*
- [hydrate](./hydrate.md) ‚Äî single-function utility-js for deferring expensive `connectedCallback` setup until an element approaches the viewport (`hydrateOnVisible`); 200px pre-load lookahead; fire-once + auto-disconnect; opt-in per specialized-section subclass, not a BaseComponent-level abstraction *(planned)*
- [font-system](./font-system.md) ‚Äî merged substrate spec for the font catalog: `font` + `typeface` metaobjects + `utility--font-face.liquid` emitter. Walks typefaces ‚Üí fonts ‚Üí `@font-face` rules; handles static + variable-weight modes, format mapping (woff2/woff/otf/ttf), skip rules; `font-display: swap` fixed *(shipped ‚Äî retrofit)*
- [modifiers-manager](./modifiers-manager.md) ‚Äî class that reads and mutates the `data-modifiers` attribute on a target element; lazy-instantiated per theme-* element via `BaseComponent` + singleton on `documentElement` *(shipped ‚Äî retrofit)*
- [events-manager](./events-manager.md) ‚Äî instance-per-element bookkeeping wrapper around `addEventListener` / `removeEventListener`; tracks attachments by element reference; `clear()` detaches everything (BaseComponent.disconnectedCallback contract); options-carried-verbatim, no-dedup-on-add semantics *(shipped ‚Äî retrofit)*
- [observers-manager](./observers-manager.md) ‚Äî instance-per-element wrapper around `ResizeObserver` / `IntersectionObserver` / `MutationObserver`; strict 3-type whitelist; per-type construction routing (Intersection takes options-on-construct, Resize/Mutation take options-on-observe); v3.0.0 dropped per-type sugar ‚Äî canonical `add(element, type, handler, options)` form *(shipped ‚Äî retrofit)*
- [cache-manager](./cache-manager.md) ‚Äî instance-per-element class memoizing values by `(type, key)` pair with optional per-entry TTL; `dom` cache type pre-initialized for the common `closest()`/`querySelector()` memoization pattern (used by `BaseComponent.section`); selective or full `clear()` *(shipped ‚Äî retrofit)*
- [utils](./utils.md) ‚Äî three pure utilities: `getRootFontSize()` for px‚Üîrem conversion, `throttle(cb)` (rAF-based, latest-args coalescing) and `debounce(cb, delay=100)` ‚Äî both expose `.cancel()` for component-disconnect parity *(shipped ‚Äî retrofit)*
- [dom](./dom.md) ‚Äî lazy-getter catalog of document-level structural elements; v2.0.0 trimmed to just `pageContent` (`<main id="page_content">`); other entries re-add when their corresponding sections ship with confirmed IDs *(shipped ‚Äî retrofit)*
- [document-utils](./document-utils.md) ‚Äî three eager document-level singletons extracted from core.js in v1.1.0: `documentModifiers` (ModifiersManager bound to documentElement), `documentScroll` (freeze-via-inset scroll-lock with restore), `documentScrollbar` (width tracking + `--scrollbar-width` CSS variable emission via lazy ResizeObserver) *(shipped ‚Äî retrofit)*
- [core](./core.md) ‚Äî JS entry point: imports + re-exports every `@theme/*` module, populates `window.Token` namespace (renamed from `window.theme` in v1.3.0 for collision safety), fires forward-looking `theme:ready` event on DOMContentLoaded; loading triggers custom-element registration as side effect *(shipped ‚Äî retrofit)*
- theme-events ‚Äî typed cross-component event bus (Bucket B dependency). `CartUpdateEvent` / `VariantUpdateEvent` carrying `detail.resource` + affected `sections`; deferred from `events-manager` (which covers add/remove of platform listeners only) and `modifiers-manager` (out-of-scope deferral). Lands when the dynamic-runtime cluster ships *(planned)*
- [container-style](./container-style.md) ‚Äî named container variant (handles project-seeded; no canonical set); handle drives a centralized `[data-modifiers*='container-style:<handle>']` rule in `layer-theme.css` scoped across `group` / `columns` / `media` *(shipped ‚Äî retrofit)*
- [spacing](./spacing.md) ‚Äî named spacing token catalog with mobile/desktop px pairs; auto-emits `--spacing-<handle>` via `utility--css-variables` (mobile in `:root`, desktop in nested `@media`); T-shirt handles (`xs`/`sm`/`md`/`lg`/`xl`) override `layer-base.css` substrate defaults via cascade position; consumed by section `block_rhythm` setting and by any component reading the unified spacing namespace *(shipped ‚Äî retrofit)*
- [content-width](./content-width.md) ‚Äî named max-inline-size catalog; emitted as `--content-width: <value>px` per-section / per-block via dynamic style; cascades through nested block-level overrides; `125rem` substrate fallback in `layer-theme.css` covers unset case; `min(var(--content-width), 100% - 2 * --gutter)` clamp in the bleed-grid named-line cap *(shipped ‚Äî retrofit)*
- [button-style](./button-style.md) ‚Äî schema-light named button variant catalog; `system.handle` drives `[data-modifiers*='button-style:<handle>']` selectors in `snippets/button.liquid`'s `{% stylesheet %}` block; 3√ó3 family/variant matrix (solid/outline/link √ó primary/secondary/tertiary); CSS authored centrally, schema stays flat *(shipped ‚Äî retrofit)*
- [media-size](./media-size.md) ‚Äî named media sizing catalog; three modes via `type` field (`ratio` / `relative` / `fixed`) plus one handle-routed special (`fill` ‚Äî reserved, load-bearing); 9 seed entries cover common compositions; mode dispatch lives in `utility--media-sizing.liquid`, shared by `media` and `embed` *(shipped ‚Äî retrofit)*

## Layer 0 ‚Äî Snippets

- [image](./image.md) ‚Äî responsive Shopify image renderer with optional mobile art direction (`<picture>` wrapping with mobile `<source>`); exposes loading / fetchpriority / preload knobs; two width ladders *(shipped ‚Äî retrofit)*
- [video](./video.md) ‚Äî Shopify-hosted video renderer wrapping `<video>` in `<media-video>`; atmosphere vs content playback profiles; mp4 source extraction from `video.sources`; mobile art-direction via second `<source>` *(shipped ‚Äî retrofit)*
- [icon](./icon.md) ‚Äî inline SVG icon loader from `assets/icon-*.svg` with dual-API (metaobject ref OR file_name string); injects `aria-hidden="true"` + optional `data-preset` (escaped); empty-on-missing via `utility--inline-asset` *(shipped ‚Äî retrofit)*
- [star-rating](./star-rating.md) ‚Äî 5-star visual using `icon-star` full/half/empty presets; resolves a numeric 0‚Äì5 rating into per-star presets via 0.5 increments; carries its own SR-only label *(planned)*
- [price-with-compare](./price-with-compare.md) ‚Äî current price + optional compare-at (strikethrough) + optional save pill; emits raw cents + currency ISO in `data-*` for a JS update path on variant change; SR-only label combines the price story *(planned)*
- [inventory-status](./inventory-status.md) ‚Äî variant inventory state pill (in-stock / low-stock / pre-order / out-of-stock); resolves four states from Shopify variant fields, emits `data-modifiers="state:<value>"`, colors per state from theme palette + scheme muted token *(planned)*
- [pagination](./pagination.md) ‚Äî two-mode pagination: numbered (SEO-friendly anchors, `<nav>` landmark, ellipsis-collapse window) or load-more (button + `aria-live` progress, JS append via `pagination.js`); shared `data-modifiers="type:<mode>"` styling surface *(planned)*
- [badge](./badge.md) ‚Äî generic tone-colored pill (label + optional icon); 6 tones (neutral / success / warning / error / info / accent ‚Äî mixed palette + scheme-role sourcing) √ó 2 styles (tint via `color-mix`, solid); per-tone CSS centralized in the snippet's stylesheet *(planned)*
- [article-card](./article-card.md) ‚Äî blog article tile (image / date / title / excerpt / author) with `::after` pseudo-overlay making the whole card clickable; opt-in line-clamp via `excerpt-clamp` modifier; localized date + responsive image *(planned)*
- [form-field](./form-field.md) ‚Äî atomic field: label + control + help + error with full ARIA wiring (`for`/`id`, `aria-describedby`, `aria-invalid`, `aria-required`, `role="alert"`); 9 types incl. text-family / textarea / select / checkbox; variable-shadow override for error-state border *(planned)*
- [payment-icons-strip](./payment-icons-strip.md) ‚Äî row of accepted payment-method icons from `shop.enabled_payment_types`; group-level `aria-label` + per-icon `aria-hidden` (one announcement, not per-method enumeration); wrap-friendly flex layout, no color overrides (brand SVGs carry their own paint) *(planned)*
- [tooltip](./tooltip.md) ‚Äî native-`popover` toggletip; non-interactive description content (`role="tooltip"`); zero-JS base (click / tap / Enter / Space toggles) + JS enhancement for hover / focus reveal + placement until CSS anchor positioning is baseline; icon or text trigger; AA touch-target floor with per-consumer AAA opt-in *(planned)*
- [quote](./quote.md) ‚Äî `<blockquote>` primitive with optional decorative opening-mark (`::before` Unicode glyph, `data-modifiers="marks:visible"`) and optional `<cite>` (4-cell `cite_text` √ó `cite_url` truth table covering attribute-only / visible-text / anchored variants); paragraph-wraps content for substrate typography; sub-component for `pull-quote` + `testimonial` L1 wrappers *(planned)*

## Layer 1 ‚Äî Theme blocks

- [rating](./rating.md) ‚Äî merchant-composable block for product rating display; metafield-first resolution with manual override-as-license + product picker for editorial placements; consumes `star-rating` L0; emits `AggregateRating` JSON-LD via `utility--json-ld` *(planned)*
- [spacer](./spacer.md) ‚Äî empty vertical-spacer block with fixed or breakpoint-paired sizing, optional `theme_color` background *(shipped ‚Äî retrofit)*
- [separator](./separator.md) ‚Äî hairline `<hr>` divider with optional content_width cap and `theme_color`-driven line color (defaults to `--color-role-border`) *(shipped ‚Äî retrofit)*
- [title](./title.md) ‚Äî heading primitive (`h1`‚Äì`h6` / `p`) with `text_style` override, optional leading icon, alignment, content_width cap, `theme_color` foreground *(shipped ‚Äî retrofit)*
- [richtext](./richtext.md) ‚Äî long-form rich-text body wrapper always emitting the `prose` utility modifier; content_width drives readability width; `theme_color` foreground *(shipped ‚Äî retrofit)*
- [button](./button.md) ‚Äî call-to-action primitive rendering `<a>` or `<button type="button">` depending on `link`; styled via the 3√ó3 `button_style` family/variant matrix; optional icon, content_width, top spacing *(shipped ‚Äî retrofit)*
- [media](./media.md) ‚Äî image or video container with overlay-content support; sizing via `media_size` (fill/ratio/relative/fixed); art direction; bleed (shared off-center-track edge case documented); image_fit cover/contain; container_style; color_scheme override; narrow overlay-content whitelist (`title`, `richtext`, `button`, `group`) *(shipped ‚Äî retrofit)*
- [embed](./embed.md) ‚Äî third-party video iframe for YouTube and Vimeo; URL parsing for 7 supported shapes (incl. Vimeo unlisted-hash); lazy-loaded; sizing via shared `media_size` utility; editor-only diagnostic for unparseable URLs *(shipped ‚Äî retrofit)*
- [group](./group.md) ‚Äî flex container with direction / stack-below (container-query driven) / alignment / gap / bleed / container-style / color-scheme override; recursive composition via explicit whitelist of the 9 L1 block types *(shipped ‚Äî retrofit)*
- [columns](./columns.md) ‚Äî CSS Grid container with seven preset ratios (2 / 3 / 4 / 1-2 / 2-1 / 1-3 / 3-1); stack-below via `@container` queries against own width; sticky-track support for 2-track layouts; bleed (shared footgun); container_style; color_scheme override; recursive nesting *(shipped ‚Äî retrofit)*

## Section host
